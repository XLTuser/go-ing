环境要求：
=============
 - PHP：80（向下兼容到71）
 - 伪静态：纯静态

安装方式：
=============
 - 1：下载压缩包上传至服务器
 - 2：解压压缩包（无需更改）
 - 3：配置域名伪静态为纯静态
